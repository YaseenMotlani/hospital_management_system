🚀 <strong><u>Hospital Management Software</u></strong><br/>
💻 <strong><u>Note:</u></strong> This application is designed for <b>desktop/laptop use only</b> and is intended for <b>hospital staff and doctors</b> to manage operations efficiently.<br/>

Hospital Management Software is a full-stack web application designed to manage hospital operations efficiently.
It includes modules for doctors, patients, appointments, inventory, and basic AI assistant support.

✨ <strong><u>The application supports</u></strong><br/>
-🔐 Secure authentication system (Login / Signup)<br/>
-👨‍⚕️ Doctor management (CRUD operations)<br/>
-📊 Dashboard overview<br/>
-📅 Appointment management<br/>
-💊 Medicine inventory tracking<br/>
-📚 Education content module<br/>
-🤖 AI Assistant (basic integration)<br/>

🎯 <strong><u>My Contribution</u></strong><br/>
-🔐 Implemented complete JWT Authentication system (Login / Signup)<br/>
-👨‍⚕️ Developed Doctor Module (Add, Update, Delete, View Doctors)<br/>
-🛡️ Protected Routes using middleware (Frontend + Backend)<br/>
-🤖 Integrated basic AI Assistant module<br/>

✨ <strong><u>Key Features</u></strong><br/>
-🔐 JWT-based Authentication & Authorization<br/>
-👨‍⚕️ Doctor Management System (CRUD APIs)<br/>
-📊 Dashboard with activity overview<br/>
-📅 Appointment tracking system<br/>
-💊 Medicine inventory management<br/>
-📱 Responsive UI design<br/>
-⚡ REST APIs using Express.js<br/>

🛠 <strong><u>Tech Stack</u></strong><br/>
<b><u>Frontend:</u></b><br/>
-React (Vite)<br/>
-JavaScript (ES6+)<br/>
-Bootstrap + CSS<br/>
-React Router DOM<br/>
-Chart.js<br/>

<strong><u>Backend:</u></strong><br/>
-Node.js<br/>
-Express.js<br/>
-MongoDB<br/>
-Mongoose<br/>
-JWT Authentication<br/>
-bcryptjs<br/>

🔐 <strong><u>Authentication Flow</u></strong><br/>
-User registers or logs in<br/>
-JWT token generated on backend<br/>
-Token stored in localStorage<br/>
-Protected routes accessed using Bearer Token<br/>

👨‍⚕️ <strong><u>Doctor Module</u></strong><br/>
-Add new doctor<br/>
-Update doctor details<br/>
-Delete doctor<br/>
-View all doctors<br/>

⚙️ <strong><u>Environment Variables</u></strong><br/>
-Create a .env file inside Backend:<br/>
-PORT=3000<br/>
-DATABASE_URL=your_mongodb_url<br/>
-JWT_SECRET=your_secret_key<br/>

🚀 <strong><u>Installation & Setup</u></strong><br/>
<strong>1) Clone Repository:</strong><br/>
-git clone https://github.com/YaseenMotlani/hospital_management_system.git
<br/>
-cd hospital-management-system<br/>

<strong>2) Backend Setup:</strong><br/>
-cd Backend<br/>
-npm install<br/>
-npm start<br/>

<strong>3) Frontend Setup:</strong><br/>
-cd Frontend<br/>
-npm install<br/>
-npm run dev<br/>
